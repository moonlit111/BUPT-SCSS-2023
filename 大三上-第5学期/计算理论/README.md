我们培养方案是计算理论、安全法、安全通论三选一。我选了计算理论。听说24级开始网安只有安全法能选。计算理论和安全通论只有信安能选。

孙岩炜老师上的，是三选一中唯一一个不用pre的课程。

如果有这门课可以选，推荐选择，给分非常高。大家都是接近满分。
