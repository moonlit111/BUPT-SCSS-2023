
第一题（原题）
HTML：
```html
<div id="d0">
  <span class="hw s0">Hello world 1</span><br>
  <span class="hw s1">Hello world 2</span><br>
  <span id="s3">Hello world 3</span><br>
  <span id="s4">Hello world 4</span><br>
  <span id="s5">Hello world 5</span>
</div>
```
（1）div 下所有文本红色
（2）Hello world 1、2 绿色
（3）Hello world 3 之后 span 橙色
参考答案
```js
（1）DOM 操作
var div = document.getElementById("d0");
var spans = div.getElementsByTagName("span");
for (var i = 0; i < spans.length; i++) {
  spans[i].style.color = "red";
}

（2）DOM 操作
var hw = document.getElementsByClassName("hw");
for (var i = 0; i < hw.length; i++) {
  hw[i].style.color = "green";
}

（3）CSS 组合选择器
#s3 ~ span {
  color: orange;
}

```
 第二题（DOM 获取元素 + style 修改）
 题目
```html
<body>
  <div id="box">
    <h3>Title</h3>
    <p>Paragraph 1</p>
    <p class="c1">Paragraph 2</p>
    <p class="c1">Paragraph 3</p>
  </div>
</body>
```
（1）第一步，请使用 **DOM 操作**，将 `div` 中所有 `p` 元素的文本颜色设置为黑色（black）。
（2）第二步，请使用 **DOM 操作**，将 class 为 `c1` 的段落文本颜色设置为蓝色（blue）。
（3）第三步，请使用 **DOM 操作**，将标题 `Title` 的文本内容改为 `"DOM Test"`。

参考答案
```js
// （1）
var box = document.getElementById("box");
var ps = box.getElementsByTagName("p");
for (var i = 0; i < ps.length; i++) {
  ps[i].style.color = "black";
}
// （2）
var c1s = document.getElementsByClassName("c1");
for (var i = 0; i < c1s.length; i++) {
  c1s[i].style.color = "blue";
}
// （3）
box.getElementsByTagName("h3")[0].innerHTML = "DOM Test";
```
第三题（CSS 后代 / 子元素选择器）
题目
```html
<body>
  <ul id="nav">
    <li><a href="#">Home</a></li>
    <li>
      <a href="#">Product</a>
      <ul>
        <li><a href="#">Phone</a></li>
        <li><a href="#">Pad</a></li>
      </ul>
    </li>
  </ul>
</body>
```
（1）请使用 CSS，使所有 `a` 元素的文本颜色为灰色（gray）。
（2）请使用 CSS，使 **nav 中第一层的 a 元素** 文本颜色为红色（red）。
（3）请使用 CSS，使子菜单中的 `a` 元素文本颜色为蓝色（blue）。
参考答案
```css
/* （1） */
a {
  color: gray;
}

/* （2） 只选 nav 的直接子元素里的 a */
#nav > li > a {
  color: red;
}

/* （3） 子菜单里的 a */
#nav ul a {
  color: blue;
}
```
第四题（伪元素 + 内容生成）
题目
```html
<body>
  <p id="p1">first paragraph</p>
  <p id="p2">second paragraph</p>
</body>
```
（1）请使用 CSS，在每个段落文本前自动添加字符串 `"* "`。
（2）请使用 CSS，将 `p1` 的首字母设置为红色（red）。
（3）请使用 CSS，使 `p2` 的首行文本设置为蓝色（blue）。

第六题（CSS 特殊性）
题目
```html
<body>
  <div id="box">
    <p class="t">Hello</p>
  </div>
</body>
```
CSS：
```css
p { color: black; }
#box p { color: blue; }
div#box .t { color: red; }
```
（1）最终 Hello 显示为什么颜色？
（2）请说明原因（从 CSS 特殊性角度）。
（3）如果想让 Hello 显示为蓝色，请写一条新的 CSS 规则（不改 HTML）。
参考答案
（1）
```text
红色
```
（2）
```text
div#box .t 的特殊性最高（ID + 类 + 元素），覆盖其它规则
```
（3）
```css
#box p {
  color: blue !important;
}
```
第七题（DOM：getElementById + getElementsByTagName + innerHTML）
题目
```html
<body>
  <div id="d1">
    <span>Apple</span><br>
    <span>Banana</span><br>
    <span id="sp3">Orange</span><br>
    <span>Grape</span>
  </div>
</body>
```
（1）第一步，请使用 DOM 操作，将 `d1` 中所有 `span` 的文本颜色设置为棕色（brown）。
（2）第二步，请使用 DOM 操作，将 id 为 `sp3` 的文本颜色设置为红色（red）。
（3）第三步，请使用 DOM 操作，将第 2 个 `span` 的文本内容改为 `"Pear"`。
参考答案
```js
// （1）
var d1 = document.getElementById("d1");
var spans = d1.getElementsByTagName("span");
for (var i = 0; i < spans.length; i++) {
  spans[i].style.color = "brown";
}
// （2）
document.getElementById("sp3").style.color = "red";
// （3）第2个span（下标1）
spans[1].innerHTML = "Pear";
```
第八题（CSS：类选择器 + ID 选择器 + 后代选择器）

```html
    <div>
      <p class="c">P2</p>
    </div>
    <p>P3</p>
  </div>
</body>
```
（1）请使用 CSS，使所有 `p` 的文字颜色为灰色（gray）。
（2）请使用 CSS，使 `#box` 内 class 为 `c` 的段落颜色为蓝色（blue）。

第九题（CSS：相邻兄弟选择器 + 后继兄弟选择器）
题目
```html
<body>
  <div id="a">A</div>
  <p class="x">X1</p>
  <p class="x">X2</p>
  <p class="x">X3</p>
</body>
```
（1）请使用 CSS，使所有 `.x` 文字颜色为黑色（black）。
（2）请使用 CSS，使紧跟在 `#a` 后的第一个 `.x`（X1）文字颜色为红色（red）。
（3）请使用 CSS，使 `#a` 后面所有 `.x`（X1、X2、X3）文字颜色为蓝色（blue）。

```css
/* （1） */

#a + .x { color: red; }

#a ~ .x { color: blue; }
```
第十题（伪类：:hover + :link/:visited）
题目
```html
<body>
  <a id="a1" href="https://example.com">Link</a>
  <p id="p1">Text</p>
</body>
```
（1）请使用 CSS，使所有超链接默认颜色为蓝色（blue）。
（2）请使用 CSS，使鼠标悬停在链接上时颜色变为红色（red）。
（3）请使用 CSS，使访问过的链接颜色为紫色（purple）。
参考答案
```css
/* （1）默认（未访问也会匹配到 link） */
a:link { color: blue; }
/* （2）悬停 */
a:hover { color: red; }
/* （3）访问后 */
a:visited { color: purple; }
```
> 注意：写伪类时一般按 link/visited/hover/active 的顺序更稳（老师常提醒这个）。

第十一题（伪元素：::before / ::after + content）
题目
```html
<body>
  <p class="msg">Hello</p>

```
（1）请使用 CSS，使所有 `.msg` 的文字颜色为黑色（black）。

第十二题（DOM + class：setAttribute / className 的基础用法）
题目
```html
<body>
  <div id="d0">
    <p id="p1">A</p>
    <p id="p2">B</p>
  </div>
</body>
```
（1）第一步，请使用 DOM 操作，将 `p1` 的文字颜色设置为红色（red）。
（2）第二步，请使用 DOM 操作，给 `p2` 添加 class 名称 `"c1"`。
（3）第三步，请编写 CSS，使 class 为 `c1` 的段落文字颜色为蓝色（blue）。
参考答案
```js
// （1）
document.getElementById("p1").style.color = "red";
// （2）两种写法都可，按老师讲过的用法选一种
document.getElementById("p2").className = "c1";

```
```css
/* （3） */
.c1 { color: blue; }
```
万能模板（考试直接套）

下面这份就是你要的“**三小问代码题万能模版**”，你拿到任何类似题，基本照抄结构改 id/class/标签名即可。
 ① DOM 批量设置样式（常用于“第一步：全变某颜色”）
```js
var box = document.getElementById("容器ID");
var els = box.getElementsByTagName("标签名");
for (var i = 0; i < els.length; i++) {
  els[i].style.样式属性 = "值";
}
```
例：把容器里所有 span 变红：

```js
var box = document.getElementById("d0");
var spans = box.getElementsByTagName("span");
for (var i = 0; i < spans.length; i++) {
  spans[i].style.color = "red";
}
```
② DOM 精准选中一类/一个（常用于“第二步：挑出某几个改色/改字”）
选中 id
```js
document.getElementById("某ID").style.color = "green";
document.getElementById("某ID").innerHTML = "新文本";
```
选中 class（批量）
```js
var els = document.getElementsByClassName("某class");
for (var i = 0; i < els.length; i++) {
  els[i].style.color = "green";
}
```
③ CSS 选择器第三问万能套路（老师最爱考）
1）后代 / 子元素
```css
#id 标签 { }      /* 后代（里面所有层级） */
#id > 标签 { }    /* 直接子元素（只下一层） */
```
2）相邻兄弟 / 后继兄弟（你老师那题同款）
```css
#某ID + 标签 { }  /* 紧挨着的下一个 */
#某ID ~ 标签 { }  /* 后面所有同级兄弟 */
```
3）伪类/伪元素
```css
a:hover { }
a:visited { }
p::first-letter { }
p::before { content:""; }
p::after { content:""; }
```
④ 答题顺序小技巧（很稳）
* 第一问（全体）用 DOM 先“铺底色”
* 第二问（局部）用 DOM 再“覆盖”
* 第三问（范围规则）用 CSS 选择器“自动选中一片”
