徐国胜上的，给分挺好。
分数构成：平时40期末60，无期中。
平时成绩主要是课堂练习（徐老师的班才会有，第三节课进行）和作业（想平时分拉满作业需要满分，老师说过作业正确率影响平时分的高低）

这门课其实是上数论与近世代数。前半学期主要学数论，难度不是很大，一般理解起来不是很难，主要都是同余二次互反等计算。后半学期学近世代数，也就是群环域的内容，这部分理解起来比较困难，也很抽象，也是比较头疼的问题。

由于近世代数部分在期中之后开始上，内容多时间紧，老师上课讲不来这么多，都是以念PPT为主。
但是近世代数内容考的不是很深，大多数也是考群的相关证明（我们当时这部分证明题只考了一道：两个正规子群的交集是正规子群）还有一些概念（比如叫你解释一下什么是整环，什么是域）。

信数期末考试会考15分概念解释，一定要牢记一些相关概念定义。一定要听最后一节复习课，老师当时提到的一些概念都考了。
这门课考原题的概率很大，老师也提到过期末考试会出很多课件中原题，因此把课件每个证明题弄明白很关键。
特别是近世代数部分，你就把书合上，每道证明题保证都能写出步骤就可以应付考试了~

这门课还是很重要的，我学这数论的时候总感觉又学了一遍离散数学，这门课也是之后现代密码学课程的基础。
